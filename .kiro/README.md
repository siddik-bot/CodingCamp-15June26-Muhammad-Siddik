# 🚀 Life Dashboard


A simple productivity dashboard to help users organize daily activities.

Built for Coding Camp project assignment.


## 🌐 Live Demo

https://github.com/siddik-bot/CodingCamp-15June26-Muhammad-Siddik


## 📌 Features


### Greeting System

- Display current time
- Display current date
- Dynamic greeting message


### ⏱ Focus Timer

Pomodoro timer:

- 25 minutes
- Start
- Stop
- Reset


### 📝 To Do List

Features:

- Add task
- Edit task
- Delete task
- Mark completed
- Save using Local Storage


### 🔗 Quick Links

Save and access favorite websites quickly.



## ✨ Additional Challenges


Implemented:

✅ Dark Mode

✅ Custom Username Greeting

✅ Prevent Duplicate Tasks

- HTML5
